# 🎲 Dice Roller 

A modern interactive **Dice Roller web application** built using **HTML, CSS and Vanilla JavaScript**.

The app simulates rolling a dice, displays the result with a visual dice face, and keeps track of the roll history.

---

## 🚀 Live Demo

👉 https://johnyisbackk.github.io/js-dice-roller/


---

## ✨ Features

- 🎲 Random dice roll (1–6)
- 🎞️ Dice roll animation
- 📜 Roll history tracking
- 🧹 Clear history button
- 🔢 Dynamic roll counter
- 💡 Empty state when no rolls exist
- 📱 Responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

No frameworks or libraries were used.

---

## 🎮 How It Works

1. Click **Roll Dice**
2. The dice animates and generates a random number between **1 and 6**
3. The result appears visually
4. The roll is added to the **history list**
5. Use **Clear History** to reset the rolls
